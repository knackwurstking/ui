# UI

This project contains some helper functions

## Package Contents

- **assets.go**: AssetURL function for generating versioned URLs with cache-busting
- **echo-middleware.go**: EchoMiddlewareCache for setting cache headers on static assets and pages
- **logger.go**: Logger struct with colorized output and multiple log levels (Info, Warn, Error, Debug)
- **router.go**: EchoRoute helper for registering HTTP routes with Echo framework
